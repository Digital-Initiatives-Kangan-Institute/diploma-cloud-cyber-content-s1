# S1-CL1 Cloud Design and Build — session scaffold (DRAFT)

**Status:** DRAFT working scaffold, 2026-05-31. Bookend sessions per Tim; the teaching/practice/AT block (sessions 2–29) is not yet detailed. Nothing here is locked in.

## Structure

- **104h goal**, delivered as **32 × 3h sessions** = 96h (8h under goal — see `documentaion/delivery_process.md` § Semester 1 structure).
- Sessions run in **AM/PM pairs on the same day**: session 1 = AM, session 2 = PM (day 1); sessions 3 & 4 (day 2); etc.
- **32 sessions = 16 teaching days = 2 days/week across weeks 1–8.**

- **S1** — intro, onboarding, systems setup, access to email + teaching materials, scenario introduction.
- **S2 → S28** — the main block: all teaching, practice exercises, and **AT1 + AT2** must fit here (27 sessions). Detailed placement of AT1 and AT2 within this block is TBD — part of the main delivery-plan work.
